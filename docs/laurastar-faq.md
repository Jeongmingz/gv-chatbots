- 공통 사용/관리: 물 사용, 첫 사용, 예열, 설명서, 정품등록
- Smart 시리즈: Smart I/M/U 차이, GO+ 비교, 다리미판 바람, 물 공급
- GO+: 예열, 물 보충, 커버 호환
- Lift 시리즈: Lift/Plus/Xtra 차이, 펄스 스팀, 필터, 소음, 열판 청소
- IZZI 시리즈: IZZI/Plus/BOSS 차이, 필터, 보일러 청소, 물 부족 경고
- IGGI: 물 사용, 마개 보관, 석회질 제거, 마개 잠김, 부품
- 다리미판/커버/액세서리: 커버 호환, 보관 두께, 균형, UD 다리미판
- AS/수리: 수거, 검수기간, 제품별 포장, 유상/정상 반송
- 주문/배송/반품: 변심 반품, 모델 변경, 구성품 배송, 보상판매
- SKU/모델 코드: 상담 매뉴얼 기준 주요 모델 코드
